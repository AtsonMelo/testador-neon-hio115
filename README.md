# Testador NEON HIO115

Projeto de bancada para testar o CLP HI Tecnologia NEON com módulo HIO115.

## Objetivo

Criar uma bancada simples para testar:

- Saídas digitais DO00 a DO03
- Entradas digitais DI00 a DI07
- Entradas analógicas AI00 a AI02
- Comunicação serial futuramente

## Etapa atual

Teste inicial da saída digital DO00 usando ST.

## Hardware disponível

- NEON com CPU450
- Módulo HIO115 no slot 1
- Fonte 24 Vcc
- Protoboard
- Resistores
- LEDs
