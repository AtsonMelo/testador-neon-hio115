# Testador NEON HIO115

Projeto de bancada para testar entradas digitais, saídas digitais e entradas analógicas do CLP HI Tecnologia NEON com módulo HIO115.

## Etapa atual

Teste inicial da saída digital DO00.

## Hardware inicial

- NEON com CPU450
- Módulo HIO115 no slot 1
- Fonte 24 Vcc
- Protoboard
- Resistores
- LEDs
